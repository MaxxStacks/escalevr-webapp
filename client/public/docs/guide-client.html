Guide d'utilisation pour les clients

Comment gérer vos véhicules et suivre vos services

Version 1.0 | Mai 2025

1. Introduction à la plateforme GestionVR

Bienvenue sur la plateforme GestionVR d'Escale VR Beaumont! Ce guide vous aidera à naviguer et utiliser efficacement notre système de gestion de services pour véhicules récréatifs. Vous y trouverez des instructions détaillées pour effectuer les tâches les plus courantes.

Note: Pour utiliser cette plateforme, vous aurez besoin d'un compte. Si vous n'avez pas encore de compte, veuillez contacter notre équipe de service client au 418-555-1234.

2. Connexion à votre compte

Pour accéder à votre compte GestionVR:

  1. Accédez à app.escale-vr.com depuis votre navigateur
  2. Cliquez sur le bouton "Se connecter" en haut à droite de la page
  3. Entrez votre nom d'utilisateur et votre mot de passe
  4. Cliquez sur "Connexion"

Si vous avez oublié votre mot de passe, cliquez sur "Mot de passe oublié?" et suivez les instructions pour le réinitialiser.

3. Gestion de vos véhicules

La section "Véhicules" vous permet de gérer tous vos véhicules récréatifs dans le système.

3.1. Ajouter un nouveau véhicule

  1. Dans le menu principal, cliquez sur "Véhicules"
  2. Cliquez sur le bouton "Ajouter un véhicule"
  3. Remplissez le formulaire avec les informations requises:
    • Marque et modèle
    • Année de fabrication
    • Numéro VIN (numéro d'identification du véhicule)
    • Notes supplémentaires (optionnel)
  4. Cliquez sur "Enregistrer"

3.2. Modifier les informations d'un véhicule

  1. Dans la liste des véhicules, cliquez sur le véhicule que vous souhaitez modifier
  2. Cliquez sur l'icône "Modifier" (crayon)
  3. Mettez à jour les informations nécessaires
  4. Cliquez sur "Enregistrer les modifications"

4. Suivi des travaux de service

La section "Services" vous permet de suivre tous les travaux de réparation ou d'entretien de vos véhicules.

4.1. Consulter les travaux en cours

  1. Dans le menu principal, cliquez sur "Services"
  2. Vous verrez une liste de tous vos travaux en cours
  3. Cliquez sur un travail pour voir les détails, y compris:
    • Description des travaux
    • État actuel
    • Date de début et date prévue de fin
    • Technicien assigné
    • Photos et notes

4.2. Comprendre les états des travaux

Chaque travail peut avoir l'un des états suivants:

État Description
Planifié Le service est prévu mais n'a pas encore commencé
En cours Les techniciens travaillent actuellement sur votre véhicule
En attente de pièces Les travaux sont en pause en attendant des pièces de rechange
En attente d'approbation Des travaux supplémentaires nécessitent votre approbation
Terminé Les travaux sont terminés et votre véhicule est prêt à être récupéré

5. Réclamations de garantie et d'assurance

Si vous avez un travail couvert par une garantie ou une assurance, vous pouvez suivre l'état de votre réclamation.

5.1. Consulter les réclamations

  1. Dans la page de détails d'un travail, consultez la section "Réclamations"
  2. Vous y verrez l'état de toute réclamation associée à ce travail
  3. Les états possibles sont: En attente, Approuvée, ou Refusée

6. Planification des rendez-vous

La section "Calendrier" vous permet de planifier de nouveaux rendez-vous pour l'entretien de vos véhicules.

6.1. Prendre un rendez-vous

  1. Dans le menu principal, cliquez sur "Calendrier"
  2. Cliquez sur "Nouveau rendez-vous"
  3. Sélectionnez le véhicule concerné
  4. Choisissez la date et l'heure souhaitées
  5. Sélectionnez le type de service requis
  6. Ajoutez une description du problème ou des travaux nécessaires
  7. Cliquez sur "Confirmer le rendez-vous"

7. Gestion de votre profil

La section "Paramètres" vous permet de gérer vos informations personnelles et vos préférences de notification.

7.1. Modifier vos informations

  1. Dans le menu principal, cliquez sur "Paramètres"
  2. Dans l'onglet "Profil", vous pouvez modifier:
    • Votre nom complet
    • Votre adresse email
    • Votre numéro de téléphone
  3. Cliquez sur "Enregistrer les modifications" pour confirmer

7.2. Configurer les notifications

  1. Dans la section "Paramètres", cliquez sur l'onglet "Notifications"
  2. Activez ou désactivez les types de notifications que vous souhaitez recevoir
  3. Cliquez sur "Enregistrer les préférences" pour confirmer

8. Obtenir de l'aide

Si vous avez besoin d'assistance supplémentaire, plusieurs options s'offrent à vous:

8.1. Page d'aide et support

  1. Dans le menu principal, cliquez sur "Aide & Support"
  2. Consultez la FAQ pour des réponses aux questions fréquentes
  3. Utilisez le formulaire de contact pour soumettre une demande d'assistance
