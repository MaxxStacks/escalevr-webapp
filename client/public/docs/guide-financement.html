Guide d'utilisation — Département Financement

Accès et responsabilités du département financement dans GestionVR

Version 1.0 | Mai 2025

1. Introduction

Ce guide est destiné aux agents du département financement d'Escale VR Beaumont. Il décrit les fonctionnalités auxquelles vous avez accès dans la plateforme GestionVR et comment les utiliser efficacement dans le cadre de votre travail.

Note : Le rôle Financement vous donne un accès spécifique à la gestion des véhicules, au calendrier et aux services. Toute modification de compte ou gestion des utilisateurs doit passer par un administrateur.

2. Connexion à votre compte

Pour accéder à la plateforme :

  1. Accédez à app.escale-vr.com depuis votre navigateur
  2. Entrez votre nom d'utilisateur et votre mot de passe
  3. Cliquez sur "Connexion"

Si vous avez des problèmes de connexion, contactez l'administrateur système.

3. Tableau de bord

Le tableau de bord vous donne un aperçu rapide de l'activité en cours :

  • Nombre de services en cours
  • Services complétés ce mois-ci
  • Véhicules enregistrés
  • Alertes et rappels de garantie

4. Gestion des véhicules

La section "Véhicules" vous permet de consulter et de gérer les informations sur les véhicules récréatifs des clients.

4.1. Consulter les informations d'un véhicule

  1. Dans le menu de gauche, cliquez sur "Véhicules"
  2. Utilisez la barre de recherche pour trouver un véhicule par marque, modèle ou NIV
  3. Cliquez sur "Voir détails" pour afficher les informations complètes

4.2. Informations disponibles par véhicule

Champ Description
Marque / Modèle / Année Identification du véhicule récréatif
NIV (numéro d'identification) Numéro unique du véhicule
Date d'activation Date de première mise en service
Date de vente Date d'achat par le client
Garantie de base Date d'expiration de la garantie du fabricant
Garantie prolongée Date de début et de fin si applicable
Couleur Couleur extérieure du véhicule
Numéro interne Numéro de référence interne Escale VR
Important : Les informations personnelles du client (adresse, téléphone, courriel) sont visibles uniquement par les agents de service et les administrateurs, conformément à la politique de confidentialité.

4.3. Garanties et alertes

Le système génère automatiquement des alertes lorsque les garanties approchent de leur date d'expiration :

  • Garantie de base : Alerte 30 jours avant l'expiration
  • Garantie prolongée : Alerte 30 jours avant l'expiration

5. Calendrier des services

Le calendrier vous permet de voir les services planifiés pour tous les véhicules.

  1. Cliquez sur "Calendrier" dans le menu principal
  2. Naviguez entre les mois à l'aide des flèches
  3. Cliquez sur un jour pour voir les services prévus ce jour-là

5.1. Codes couleur des services

Couleur Type de service
🟦 Bleu DAF (Détail, alignement, finition)
🟩 Vert PDI (Préparation à la livraison)
🟨 Jaune Garantie / Garantie prolongée
🟧 Orange Service régulier / Saisonnier
🟥 Rouge Assurance

6. Messagerie interne

Vous pouvez communiquer avec les autres membres de l'équipe via la messagerie interne.

  1. Cliquez sur "Messagerie" dans le menu de gauche
  2. Sélectionnez une conversation existante ou cliquez sur "Nouvelle conversation"
  3. Rédigez votre message et cliquez sur l'icône d'envoi
  4. Pour joindre un fichier, cliquez sur l'icône de trombone (📎)
Notifications : Vous recevrez une notification dans la cloche en haut à droite et un badge sur l'icône Messagerie lorsqu'un nouveau message vous est adressé.

7. Paramètres de votre profil

Pour modifier vos informations personnelles :

  1. Cliquez sur "Paramètres" dans le menu de gauche
  2. Dans l'onglet "Profil", modifiez votre nom, courriel ou téléphone
  3. Pour changer votre photo de profil, cliquez sur l'icône de caméra
  4. Pour changer votre mot de passe, cliquez sur l'onglet "Mot de passe"
  5. Cliquez sur "Enregistrer les modifications"
